# 🎬 Movie Directory App

A specialized React application built with **Vite** and **React Router v6** for managing film directors and their discographies. This project follows a structured routing architecture to handle complex parent-child relationships between directors and movies.

## 🚀 Features
* **Director Management:** View, add, and explore detailed profiles of world-renowned directors.
* **Movie Integration:** Dynamic nested routing for viewing specific movie details.
* **Robust Routing:** Fully implemented navigation using `react-router-dom`.
* **Data Validation:** Graceful handling of invalid IDs and missing data.

## 🛠️ Tech Stack
* **Frontend:** React 18
* **Routing:** React Router v6
* **Tooling:** Vite
* **Testing:** Vitest & React Testing Library
